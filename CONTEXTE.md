L'objectif de ce dépôt est de créer un package à partir d'un nombre restreint de fonctions issues du
dépôt EmeRaudePrev qui est consultable dans Documents/Github/EmeRaudePrev. Les fonctions en question 
sont les fonctions contenues dans 9fonctions/fonctions_pour_ols. 

Le nombre de fonctions est assez restreint. Elles concernent l'estimation d'équation à correction d'erreur.
Elles (i) calculent des tests d'autocorrélation/
heteroscedasticité/normalité (ii) calculent les valeurs critiques d'ericsson/mackinnon 
(iii) intégrent les résultats dans un tableau.
(iv) réalisent des simulations statiques 
(v) réalisent des simulations dynamiques
